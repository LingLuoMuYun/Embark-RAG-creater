# 数据结构并拢改造 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将当前重复的文档、分片、向量和候选知识模型并拢为 `DocumentSource` / `DocumentChunk` 一套主模型，并让 RAG 管理、检索、Agent、分析和前端全部消费新结构。

**Architecture:** 以 `DocumentSource` 替代 `KnowledgeDocument`，以 `DocumentChunk` 替代 `KnowledgeChunk`，保留 `KnowledgeBaseDocument` 作为知识库与文档的多对多关联表。向量直接存入 `DocumentChunk.embedding`，AI 提炼结果直接写入 `DocumentChunk.content/category/type`，删除 `ChunkEmbedding` 和 `CandidateKnowledge`。先改 schema 和服务边界，再逐层迁移 API、前端、RAG、Agent 和 analytics。

**Tech Stack:** Next.js App Router, React, TypeScript, Zod, Zustand, Prisma 7, SQLite, Tailwind CSS, shadcn/ui.

```ts
async function assertDocumentIdsExist(
  tx: Prisma.TransactionClient,
  documentIds: string[]
) {
  const uniqueDocumentIds = [...new Set(documentIds)];
  if (uniqueDocumentIds.length === 0) return uniqueDocumentIds;

  const existingDocuments = await tx.documentSource.findMany({
    where: { id: { in: uniqueDocumentIds } },
    select: { id: true },
  });

  if (existingDocuments.length !== uniqueDocumentIds.length) {
    const existingIds = new Set(existingDocuments.map((document) => document.id));
    const missingIds = uniqueDocumentIds.filter((id) => !existingIds.has(id));
    throw badRequest("some documents do not exist", { documentIds: missingIds });
  }

  return uniqueDocumentIds;
}
```

```ts
async function generateAndStoreChunkEmbeddings(documentSourceId: string) {
  const chunks = await prisma.documentChunk.findMany({
    where: { documentSourceId },
    orderBy: { chunkIndex: "asc" },
  });

  const missingEmbeddingChunks = chunks.filter((chunk) => !chunk.embedding);
  if (missingEmbeddingChunks.length === 0) return;

  const embeddings = await batchEmbedTexts(
    missingEmbeddingChunks.map((chunk) => chunk.content)
  );

  await Promise.all(
    embeddings.map((result) => {
      const chunk = missingEmbeddingChunks[result.textIndex];

      return prisma.documentChunk.update({
        where: { id: chunk.id },
        data: { embedding: JSON.stringify(result.embedding) },
      });
    })
  );
}
```
